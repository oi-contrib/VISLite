如何引入？

安装

首先，你需要使用npm安装此包： 

npm install --save vislite

全量引入

也就是引入后可以通过把柄VISLite来调用其上的方法：

ES Module 

import VISLite from 'vislite'

当然，也可以指明具体的内容，比如： 

import { Example } from 'vislite'
温馨提示：上面的Example只是一个占位符，使用的时候应该换成具体的，比如Canvas、SVG、rotate、Matrix4等，下同。

Script 标签

此外，你也可以使用Script标签引入：

或直接使用在线的CDN引入：

按需引入

在大部分情况下，你可能不会用到所有能力，所以全量引入并不总是一个好的选择： 

import Example from "vislite/lib/Example/index.es.min.js"

TypeScript 源码引入

如果你希望直接引入源码使用而不是打包后的js文件，你可以：

全量引入 

import VISLite from 'vislite/package/index'
温馨提示：这要求你的项目配置了TypeScript解析相关内容。

按需引入

同样的，按需引入例子： 

import Example from "vislite/package/Example/index"
