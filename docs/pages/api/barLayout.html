直方布局
1.4.0 新增

引入 

import { BarLayout } from "vislite"

语法

通过传递一个配置项即可创建布局对象： 

let barLayout = new BarLayout(config)

其中config表示布局求解的原始数据格式，如果你数据的格式类似下面：

一维 

{
    category: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun'],
    data: [120, 200, 150, 80, 70, 110, 130]
}

二维 

{
    category: ['一月', '二月', '三月'],
    value: [{
        name: "City Alpha",
        data: [165, 170, 30]
    }, {
        name: "City Beta",
        data: [150, 105, 110]
    }, {
        name: "City Gamma",
        data: [220, 82, 63]
    }]
}

方法

setOption

设置布局的行为： 

    barLayout.setOption(option)

其中option应该是一个json，用以规范直方图的行为，可选属性如下：

x

直方图左下角横坐标，number类型，默认50。

y

直方图左下角纵坐标，number类型，默认350。

width

直方图宽，number类型，默认400。

height

直方图高，number类型，默认300。

category

直方图分类刻度尺，字符串类型，可选值为：

  • xAxis：默认值，表示x轴作为分类轴；
  • yAxis：表示y轴作为分类轴。

duration

切换时长，number类型，默认200（即0.2s）。

use

单纯的直方图坐标计算： 

    let bar = barLayout.use(initBar)

bind

绑定新的数据和渲染方法： 

barLayout.bind(initBar, function(bar){
    // 绘制出直方图
})
温馨提示：上述形参bar的格式你可以点击 进行查阅。

unbind

解除绑定： 

    barLayout.unbind()

doUpdate

主动触发绘制更新： 

    barLayout.doUpdate()
