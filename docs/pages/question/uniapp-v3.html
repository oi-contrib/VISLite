比如你使用如下语句引入ui-canvas组件： 

    import uiCanvas from "vislite/uni-app/ui-canvas.vue";

你可能会看见如下错误提示：

Could not find a declaration file for module 'vislite/uni-app/ui-canvas.vue'.

怎么办？你可以在项目中找到 tsconfig.json 文件，然后在里面添加这样的配置： 

{
    "compilerOptions": {
        "types": [ "vislite/types" ]
    }
}

如此就不会报错了。当然，你也可以直接使用 easycom 的方式进行配置，这将不会有任何错误： 

{
    "easycom": {
        "autoscan": true,
        "custom": {
            "ui-canvas": "vislite/uni-app/ui-canvas.vue"
        }
    }
}
