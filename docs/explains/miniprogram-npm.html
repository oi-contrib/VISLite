构建 npm

在开始使用之前需要进行构建： 

    通过在微信小程序开发工具界面的『工具』->『构建 npm』

指定npm位置

如果你通过project.config.json里面的 miniprogramRoot属性修改了小程序根位置，比如：

特别提醒：修改配置后，微信开发者工具需要关闭后再重新打开。 
{
    "miniprogramRoot": "miniprogram/"
}

那么，你就需要在此文件中补充如下配置后再进行构建： 

{
    "setting": {
        "packNpmManually": true,
        "packNpmRelationList": [
          {
            "packageJsonPath": "./package.json",
            "miniprogramNpmDistDir": "./miniprogram/"
          }
        ]
    }
}
